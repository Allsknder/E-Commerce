We made a row in the top of the Modal Body, That contains three columns (images) for a specific product.
We named that row: "More Images"
1) Now we need to add field to the Product Class so the Admin can add more images about a speicific product.

2) We need to specify the number of entered images for one product to three Images, 
a - Three images other than the main one. 
b - The limit must be a simple conitional so we can remove this limit any time later.

3) Remember to: 
1) put the field in the Product Class.
2) Make a folder for the "More Images" in your static directories.
3) Tell Django where is this Media by linking up the path to the folder in  the 'settings.py' file.
4) Test this idea in multiple products.
5) Remember not to spend so much time in the Front-End Styling, All of this styling may be deleted for good.
6) Every file in the Website (No matter what is its type <Icon, Image, text file>) has:
a - URL, (To access it from the Server)
b - Path, (To its place in the project folder)
 
4) The Algorithm:
1) Doing a Nested List:
List inside of a list.
I mean Each element in the 'urlpatterns' list will have an element after it that is another list, And this list is going to be containing these more images urls.
